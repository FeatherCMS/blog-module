Let's get started

Feather is an experimental open source CMS written in Swift using the Vapor framework. Feather is designed for the people who are looking for a fast, minimalistic, simple and elegant little place in the chaotic world of the modern web. Feather provides you a SEO friendly, mobile first frontend which is 100% JavaScript FREE, still you can customize the entire look of the website by writing some basic HTML & CSS. Editors can easily maintain the contents of all the pages (including meta information) by using the web-based responsive admin interface. Developers can utilize the built-in JSON based REST API layer and they can extend the system through modules and dynamic hook functions if they want to go beyond the basics... üöÄ

The future of Feather...

I don't want to promise anything, but here are a few things that I want to build:

  • File storage - upload and manage pictures globally
  • Menu module - dynamic menu items and locations
  • Search - global search functionality
  • User ACL - access control, roles, permissions
  • SiWA - Sign in With Apple
  • Visual content editor - VCE
  • Dynamic module loader and improved module lifecycle

Feather was created by one person, so if you feel like that you could help with some of the features I described above, please don't hesitate to submit a new pull request on GitHub.

...a little back-story

Feather was born about 10 years ago written in PHP5, when two university student decided to build a modular content management system for their web-based side projects. The original name of the project was entropy-framework and it worked surprisingly well, but the authors were developers, so they had no idea how to run a marketing campaign or get investment for such a project. ü§ì

Time has passed and the project was slowly forgotten, but the original developers gained so much from this experiment. They learned how to communicate and work together and most importantly how to build simple, but well-structured modular, scalable applications.
